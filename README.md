# OpenAPI MCP Hub

Library to convert OpenAPI specifications to the Model Context Protocol (MCP) for processing by MCP client tools.

## Installation

To install the necessary dependencies, run the following Maven command:

```bash
mvn clean install
```

## Key Features

*   **OpenAPI to MCP Conversion:** Automatically converts OpenAPI 3 specifications into MCP tools, schemas, and API execution providers.
*   **Dynamic Tool Creation:** Generates tools dynamically from your existing API documentation.
*   **Extensible:** Provides a foundation for building custom MCP integrations with your APIs.
*   **Server Implementation:** Includes a sample Jetty-based server for hosting the MCP tools.
